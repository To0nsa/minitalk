Detailed Description

Sends a string message to the Minitalk server using UNIX signals.

This file implements the client-side logic of the Minitalk project. The client converts each character of a string message into bits and sends them one by one to the server using SIGUSR1 and SIGUSR2. After each bit, the client waits for an acknowledgment from the server.

The message is terminated with a null byte ('\0').
